# SMCS 사용 가이드 v1.0

> **SMCS** — 사내 CS 이슈 관리 시스템. 본 가이드 1페이지로 핵심 동작을 익힐 수 있습니다.

## 시작하기

1. 브라우저에서 `<your-smcs-domain>` 접속(운영 도메인 확정 후 즐겨찾기 권장).
2. 사용자명·비밀번호 입력 후 **로그인**. 역할에 따라 자동으로 다음 화면으로 이동합니다.
3. 첫 로그인 후 임시 비밀번호 변경은 별도 안내(관리자 발급).

> 화면 캡처는 `docs/screenshots/` 폴더 참조 (PII 마스킹 처리). 운영 도메인에서 실제 사용 화면과 동일합니다.

---

## 1. CS 접수자 (AGENT) — 이슈 등록·배정

> 화면: `docs/screenshots/01-agent-issue-list.png`, `02-agent-issue-form.png`

1. 로그인 → **이슈** 메뉴(자동 이동) → 우측 상단 **`+ 신규 등록`** 클릭 (또는 단축키 `N`).
2. 등록 폼에 **제목 / 발신자명 / 전화번호 / 내용** 입력. 카테고리는 입력 중 자동으로 제안됩니다 — 수정 가능.
3. **`Ctrl+S`** 로 저장 → 자동으로 이슈 상세 페이지로 이동합니다.
4. 상세에서 **담당자 배정** 드롭다운 → 현장 작업자 선택 → 확인. 배정 즉시 작업자에게 알림이 발송됩니다.
5. 필터/검색: 리스트 상단의 검색창·상태 필터·기간 필터로 과거 이슈를 조회할 수 있습니다.

**팁** — 우선순위 정렬이 기본입니다. URGENT(빨간 막대)가 위로 올라옵니다.

---

## 2. 현장 작업자 (FIELD) — 모바일 조치·완료

> 화면: `docs/screenshots/03-field-mobile-home.png`, `04-field-mobile-detail.png`

1. 휴대폰 브라우저에서 SMCS 접속 → 로그인 → 자동으로 **내 작업** 카드 리스트로 이동.
2. 카드 탭하여 상세 → **사진 추가** 버튼 → 카메라/갤러리 → 최대 10장(1장 10MB 이하, JPG/PNG).
3. **조치 코멘트** 입력 → "추가" → 활동 로그에 기록됩니다.
4. 작업 완료되면 **`완료 처리`** 버튼 → 확인 모달 → **확인**.
5. 검수에서 재오픈되면 다시 본인 리스트에 표시되며, 알림이 옵니다.

**팁** — 카메라 권한이 거부되면 갤러리에서 선택할 수 있습니다.

---

## 3. 관리자 (ADMIN) — 대시보드·보고서·사용자/카테고리 관리

> 화면: `docs/screenshots/05-admin-dashboard.png`, `06-admin-reports.png`, `07-bell-dropdown.png`

1. 로그인 → 자동으로 **대시보드**로 이동.
2. 상단 KPI 4 카드(어제 신규/처리/미처리/평균 처리 시간) + 카테고리 분포 차트 + 담당자별 처리량 확인.
3. **보관함** 메뉴 → 일간/주간 PDF 보고서 클릭 → 미리보기. 매일 08:00 KST · 매주 월요일 08:00 KST 자동 생성.
4. 헤더 우측 **🔔 벨 아이콘** 클릭 → 최근 10건 알림 드롭다운 → 항목 클릭 시 해당 이슈 또는 보고서로 이동(읽음 처리 자동).
5. **사용자 관리** — 추가/비활성화/역할 변경 가능. 신규 추가 시 임시 비밀번호가 1회 화면에 표시됩니다 — 사용자에게 별도 채널로 전달.
6. **카테고리 관리** — L1/L2/L3 탭에서 추가/이름 수정/순서 변경/비활성화 + 자동 분류용 키워드 편집.

**팁** — 비활성화된 카테고리는 신규 등록 폼에서 숨겨지지만, 과거 이슈는 영향받지 않습니다.

## 트러블슈팅

| 증상 | 원인 | 해결 |
| :--- | :--- | :--- |
| **로그인 실패가 반복됨** | 5회 실패 시 10분 자동 잠금 | 카운트다운 종료 후 재시도. 비밀번호 분실은 관리자에게 재발급 요청. |
| **알림 뱃지가 갱신되지 않음** | 시스템은 30초 주기로 자동 확인(보이는 탭만). 백그라운드 탭은 일시 중단. | 탭을 활성화하거나 새로고침(`Ctrl+R`). 30초 이상 지나도 미반영 시 관리자에게 문의. |
| **모바일 사진 첨부 실패** | (a) 파일 크기 10MB 초과, (b) JPG/PNG 외 형식, (c) 한 이슈에 이미 10장 첨부됨 | 사진 크기 축소(휴대폰 카메라 설정에서 해상도 낮춤) 또는 기존 첨부 일부 삭제. |

## 단축키 (데스크탑)

| 단축키 | 동작 |
| :----- | :--- |
| `N` | 신규 이슈 등록 (이슈 리스트에서) |
| `Ctrl+S` | 폼 저장 (등록·편집 화면) |
| `Esc` | 모달 취소 |
| `Tab` / `Shift+Tab` | 다음/이전 필드 이동 (마우스 없이 전체 흐름 가능) |

---

**문의/오류 신고**: 관리자(`admin1` 또는 사내 IT 담당자). SMCS 운영 가이드는 `docs/operations.md` 참조(운영자용).
