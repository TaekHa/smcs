# 11. Responsive Breakpoints

| 디바이스 | 폭 | 적용 |
|---------|-----|------|
| 모바일 (소) | 320 ~ 575 | 단일 컬럼, 카드 스택 |
| 모바일 (대) / 태블릿 (세로) | 576 ~ 767 | 동상, 일부 2컬럼 |
| 태블릿 (가로) | 768 ~ 991 | 데스크톱 레이아웃 적용 시작 |
| 데스크톱 | 992 ~ 1199 | 표준 |
| 데스크톱 (대) | 1200 ~ 1599 | 표준 + 여백 |
| 데스크톱 (초대) | 1600+ | 최대 폭 1440px로 제한 + 가운데 정렬 |

**모바일 ↔ 데스크톱 전환:**
- 같은 코드베이스, Ant Design 반응형 grid (`<Row gutter>`, `<Col xs sm md lg xl>`)
- `/m/*` 경로는 **모바일 전용 화면** — 데스크톱에서도 동작은 하지만 컨텐츠 폭 480px로 제한
