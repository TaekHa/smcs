# 5. Visual Design System

## 5.1 색상 팔레트 (Ant Design 5 기본 + 시맨틱)

| 토큰 | Hex | 사용처 |
|------|-----|--------|
| **Primary** | `#1677ff` | 주 액션 버튼, 링크, 활성 상태 |
| **Success** | `#52c41a` | 완료/검수 상태, 성공 토스트 |
| **Warning** | `#fa8c16` | HIGH 우선순위, 경고 |
| **Danger** | `#ff4d4f` | URGENT 우선순위, 삭제, 에러 |
| **Info** | `#13c2c2` | NEW 상태 |
| **Purple** | `#722ed1` | VERIFIED 상태 |
| **Gray-9** | `#262626` | 본문 텍스트 |
| **Gray-6** | `#8c8c8c` | 보조 텍스트, LOW 우선순위 |
| **Gray-4** | `#d9d9d9` | 보더 |
| **Gray-2** | `#fafafa` | 배경 (테이블 헤더) |

**우선순위 색상 매핑 (PRD FR16 일치):**
```
URGENT  ━━━━━  #ff4d4f  🔥  긴급
HIGH    ━━━━━  #fa8c16  ⚠️  높음
NORMAL  ━━━━━  #1677ff  •   보통
LOW     ━━━━━  #8c8c8c  ○   낮음
```

**상태 색상:**
```
NEW         #13c2c2  접수
ASSIGNED    #1677ff  배정
IN_PROGRESS #fa8c16  진행중
DONE        #52c41a  완료
VERIFIED    #722ed1  검수
```

## 5.2 타이포그래피

| 요소 | 폰트 | 크기 | 굵기 | 행간 |
|------|------|------|------|------|
| 페이지 제목 | Pretendard / Noto Sans KR | 24px | 600 | 1.4 |
| 섹션 제목 | 동상 | 18px | 600 | 1.5 |
| 본문 | 동상 | 14px | 400 | 1.6 |
| 보조 | 동상 | 12px | 400 | 1.5 |
| 모바일 카드 제목 | 동상 | 16px | 500 | 1.4 |
| 단축키 표시 | monospace | 12px | 500 | - |

> Ant Design 기본 폰트 스택을 따른다. 한국어는 Pretendard 또는 Noto Sans KR.

## 5.3 스페이싱 (8pt grid)

`4, 8, 16, 24, 32, 48, 64` — 모든 여백/간격은 이 값들 중 하나.

## 5.4 보더 / 라운드

- 라운드: **6px** (Ant Design 기본)
- 카드 보더: `1px solid #f0f0f0`
- 그림자: `0 2px 8px rgba(0,0,0,0.08)` (호버), `0 4px 16px rgba(0,0,0,0.12)` (모달)

## 5.5 아이콘

- Ant Design Icons (개별 import — 트리 셰이킹)
- 우선순위 이모지는 텍스트 폴백
- 활동 로그 아이콘:
  - 📝 코멘트 / 🎯 배정 / 🔄 상태 변경 / 📷 첨부 / ✅ 완료 / 🔁 재오픈
