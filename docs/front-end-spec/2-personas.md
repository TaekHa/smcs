# 2. Personas

## 2.1 페르소나 #1 — 박지영 (32세, CS 접수자)

| 항목 | 내용 |
|------|------|
| **직무** | CS팀 주임. 1년 6개월 경력. |
| **하루** | 9:00-18:00, 점심 1시간. 통화 30~50건/일 |
| **환경** | 헤드셋 통화. 24인치 듀얼 모니터. 키보드 위주. |
| **습관** | 단축키 사랑함. 동시에 여러 일 처리. 통화 끊으면 즉시 정리해서 잊지 않음. |
| **목표** | "통화 중에는 듣는 데 집중하고, 끊자마자 30초 안에 카드 정리하기" |
| **고민** | "지금은 엑셀에 적는데 행 추가 위치 헷갈리고, 전화번호 중복 체크가 안 됨" |
| **시스템에 바라는 것** | 빠른 입력, 자동 채움, 과거 이력 즉시 조회, 단축키 |
| **싫어하는 것** | 모달 여러 개 떠서 흐름 끊김, 페이지 이동 시 입력값 날아감 |

## 2.2 페르소나 #2 — 김민호 (38세, 현장 작업자)

| 항목 | 내용 |
|------|------|
| **직무** | 시설 기사 8년차. 아파트 단지 순회. |
| **하루** | 오전 4건, 오후 4건 평균. 차량 이동 + 현장 작업 |
| **환경** | 폰(아이폰 14). 작업 중에는 시스템 안 봄. **조치 끝나고 차에서 입력**. |
| **습관** | 사진 많이 찍음. 글은 짧게. 카톡 대화 기록처럼 시간순 정리 선호. |
| **목표** | "작업 끝나고 5분 안에 보고서 작성하고 다음 현장 가기" |
| **고민** | "지금은 카톡으로 사무실에 보고하는데 누락되거나, 다른 직원이 같은 일 또 출동" |
| **시스템에 바라는 것** | 폰에서 빠른 사진 첨부, 자동 위치(미구현 OK), 다음 할 일 우선순위 |
| **싫어하는 것** | 작은 버튼, 글자 많은 폼, 인터넷 약할 때 입력 중 끊김 |

## 2.3 페르소나 #3 — 이수진 (45세, CS팀장)

| 항목 | 내용 |
|------|------|
| **직무** | CS팀장. 10년차. CS팀+현장팀 운영 책임. |
| **하루** | 9:00-18:30. 회의/보고 많음 |
| **환경** | 27인치 모니터, 가끔 폰. 출장 잦음. |
| **습관** | 매일 아침 어제 상황 1분 안에 파악하고 싶음. 임원 보고 주 1회. |
| **목표** | "팀 처리량 트렌드 + 고객 불만 패턴을 데이터로 보고 의사결정" |
| **고민** | "월요일마다 엑셀로 보고서 만드느라 2시간씩 씀. 직원별 처리량 파악 어려움" |
| **시스템에 바라는 것** | 자동 보고서, 핵심 KPI 1화면, 부서별 비교 |
| **싫어하는 것** | 데이터 신뢰 못함 (누락된 이슈 많을 때), 화면 복잡 |
