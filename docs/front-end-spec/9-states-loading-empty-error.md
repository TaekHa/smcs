# 9. States: Loading, Empty, Error

## 9.1 Loading 상태

**원칙: 스피너 X, Skeleton O** — 사용자에게 "곧 무엇이 보일지" 미리 알려준다.

| 화면 | Loading |
|------|---------|
| 이슈 리스트 | 5행 Skeleton 행 |
| 이슈 상세 | 메타데이터 영역 + 본문 + 로그 모두 Skeleton |
| 대시보드 | KPI 카드 + 차트 Skeleton |
| 모바일 홈 | 카드 3개 Skeleton |
| 알림 드롭다운 | 3행 Skeleton |

**전역 인디케이터:**
- Ant Design `<Spin>`은 모달 내 폼 제출, 사진 업로드 등에만 사용

## 9.2 Empty 상태

| 화면 | 빈 상태 메시지 + 액션 |
|------|------------------------|
| 이슈 리스트 (전체) | "📭 아직 등록된 이슈가 없습니다." + [+ 신규 등록] 버튼 |
| 이슈 리스트 (필터 결과) | "🔍 조건에 맞는 이슈가 없습니다." + [필터 초기화] |
| 대시보드 차트 | "이 기간에 데이터가 없습니다. 다른 기간을 선택해 보세요." |
| 모바일 홈 | "🎉 현재 배정된 작업이 없습니다. 휴식하세요!" |
| 활동 로그 | (절대 빈 상태 없음 — 최소 CREATED 이벤트 존재) |
| 알림 | "🔔 새 알림이 없습니다." |
| 첨부 사진 (상세) | (영역 자체 숨김) |
| 보고서 보관함 | "📄 아직 생성된 보고서가 없습니다. 첫 보고서는 내일 아침 08:00에 자동 생성됩니다." |

## 9.3 Error 상태

**원칙: "왜" + "어떻게" 둘 다 알려준다.**

| 에러 유형 | 메시지 예시 | 액션 |
|----------|------------|------|
| 네트워크 끊김 | "⚠️ 서버에 연결할 수 없습니다. 잠시 후 다시 시도해 주세요." | [재시도] 버튼 |
| 권한 부족 (403) | "🔒 이 작업에 대한 권한이 없습니다. 관리자에게 문의해 주세요." | [홈으로] |
| 찾을 수 없음 (404) | "🔍 이슈를 찾을 수 없습니다. 삭제되었거나 잘못된 링크일 수 있어요." | [리스트로] |
| 폼 검증 실패 | 필드별 빨간 메시지 + 첫 번째 필드로 자동 스크롤 | - |
| 사진 업로드 실패 | "📷 사진 업로드 실패. 네트워크를 확인하고 다시 시도해 주세요." | [재시도] |
| 사진 크기 초과 | "📷 사진 크기가 10MB를 초과합니다. 다른 사진을 선택해 주세요." | - |
| Rate Limit (429) | "⏱ 너무 많은 요청입니다. 1분 후 다시 시도해 주세요." | (자동 대기) |
| 세션 만료 (401) | (자동 로그아웃 + 토스트) "🔐 세션이 만료되었습니다. 다시 로그인해 주세요." | - |
| 서버 에러 (500) | "🛠 일시적인 오류가 발생했습니다. 다시 시도해 주세요." (+ traceId 표시) | [재시도] |

**ErrorBoundary fallback:**
```
┌─────────────────────────────────────┐
│             🛠                      │
│   문제가 발생했어요                  │
│                                     │
│   페이지를 새로고침해 주세요.        │
│   문제가 계속되면 관리자에게 알려주세요.│
│                                     │
│   [새로고침]   [홈으로]              │
│                                     │
│   에러 ID: abc-123 (관리자 문의용)   │
└─────────────────────────────────────┘
```
