# 8. Interaction Patterns

## 8.1 단축키 시스템

| 화면 | 단축키 | 동작 |
|------|--------|------|
| **전역** | `?` 또는 `Ctrl+/` | 단축키 도움말 모달 |
| **전역** | `/` | 페이지 내 검색 포커스 |
| **전역** | `Esc` | 모달 닫기 / 취소 |
| **이슈 리스트** | `N` | 신규 등록 페이지 |
| **이슈 리스트** | `↑/↓` | 행 선택 이동 |
| **이슈 리스트** | `Enter` | 선택된 이슈 상세 |
| **등록 폼** | `Ctrl+S` / `Cmd+S` | 저장 |
| **등록 폼** | `Alt+1~4` | 우선순위 변경 |
| **카테고리 모달** | `Tab` | 컬럼 이동 |
| **카테고리 모달** | `↑/↓` | 옵션 이동 |
| **카테고리 모달** | `Enter` | 확정 |

**단축키 도움말 모달 디자인:**
```
┌─────────────────────────────────────┐
│ 단축키                       ✕ ESC │
├─────────────────────────────────────┤
│ 전역                                │
│   ?           이 도움말             │
│   Esc         취소 / 모달 닫기      │
│   /           검색                  │
│                                     │
│ 이슈 리스트                         │
│   N           신규 등록             │
│   ↑↓          행 이동               │
│   Enter       선택한 이슈 열기      │
│                                     │
│ 등록 폼                             │
│   Ctrl+S      저장                  │
│   Alt+1~4     우선순위 변경         │
└─────────────────────────────────────┘
```

## 8.2 마이크로 인터랙션 (Q5 = b)

**새 알림 도착 펄스:**
```css
@keyframes notification-pulse {
  0%   { transform: scale(1);   }
  50%  { transform: scale(1.18); }
  100% { transform: scale(1);   }
}

.bell-icon.has-new {
  animation: notification-pulse 0.6s ease-in-out 1;
}
```
- 카운트가 증가할 때마다 1회 재생
- `prefers-reduced-motion: reduce` 사용자에게는 비활성화

**상태 전이 (Steps 진행 바):**
- 다음 단계로 이동 시 새 단계 점이 부드럽게 채워짐 (300ms)

**카드 호버 (데스크톱):**
- `transform: translateY(-2px)` + 그림자 강화 (150ms)

**버튼 클릭 피드백:**
- Ant Design 기본 ripple 유지

## 8.3 자동 저장 (Draft)

**등록 폼 sessionStorage Draft:**
- 사용자 입력 5초마다 저장
- 페이지 이탈 시 (`beforeunload`) 변경사항 있으면 경고
- 다시 방문 시 "💾 작성 중이던 내용이 있습니다. 복원하시겠어요?" 토스트
- 저장 성공 시 Draft 자동 삭제
