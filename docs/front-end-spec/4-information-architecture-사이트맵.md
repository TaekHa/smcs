# 4. Information Architecture (사이트맵)

```
SMCS
├── /login                            (Public)
│
├── 📋 CS / Admin 영역 (데스크톱 우선)
│   ├── /issues                       이슈 리스트 (메인)
│   ├── /issues/new                   신규 등록
│   ├── /issues/:id                   이슈 상세
│   ├── /dashboard                    대시보드 (Admin)
│   ├── /reports                      보고서 보관함 (Admin)
│   │   ├── ?tab=daily
│   │   └── ?tab=weekly
│   └── /reports/:kind/:date          PDF 미리보기 페이지
│
├── 📱 Field 영역 (모바일 우선)
│   ├── /m                            내 작업 카드 스택
│   ├── /m/issues/:id                 이슈 상세 + 사진 + 조치
│   └── /m/notifications              알림 페이지
│
├── 🔔 공통
│   └── /notifications                알림 페이지 (데스크톱)
│
└── ⚙️ Admin 전용
    ├── /admin/users                  사용자 관리
    └── /admin/categories             카테고리 관리 (L1/L2/L3 탭)
```

## 4.1 네비게이션 구조

**데스크톱 헤더:**
```
┌─────────────────────────────────────────────────────────────────────┐
│ [SMCS]   이슈  대시보드  보고서  관리▾        🔔3   👤 박지영 ▾  │
└─────────────────────────────────────────────────────────────────────┘
```

**모바일 (현장 작업자 `/m`):**
```
┌────────────────────────┐
│  [SMCS] 김민호      🔔2 │
└────────────────────────┘
```
> 단순화 — 메뉴/네비게이션 거의 없음. 카드 스택이 곧 메뉴.
